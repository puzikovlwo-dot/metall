# Структура исходных данных

## 1. Таблица «Продажи» (sales.csv)

Содержит все сделки компании. 2000 записей за период январь — июнь 2026 года.

| № | Поле | Тип данных | Описание | Пример |
|---|---|---|---|---|
| 1 | **Дата** | Дата | Дата заключения сделки | 15.03.2026 |
| 2 | **Номер договора** | Текст | Уникальный номер договора | Д-2026-00123 |
| 3 | **ID клиента** | Число | Уникальный идентификатор клиента | 42 |
| 4 | **Название клиента** | Текст | Наименование организации или ФИО | ООО «СтальПром-042» |
| 5 | **Тип клиента** | Текст | Опт / Розница | Опт |
| 6 | **ID менеджера** | Число | Уникальный идентификатор менеджера | 3 |
| 7 | **ФИО менеджера** | Текст | ФИО менеджера | Сидоров В.Г. |
| 8 | **Категория** | Текст | Категория товара: Трубы / Лист / Арматура / Метизы | Трубы |
| 9 | **Количество** | Число | Количество единиц товара | 150 |
| 10 | **Цена за ед.** | Число | Цена за единицу товара, Br | 485,30 |
| 11 | **Сумма** | Число | Общая сумма сделки, Br | 72 795,00 |
| 12 | **Статус** | Текст | Текущий статус сделки | Оплачено |

### Статусы сделок

| Статус | Описание | Доля в данных |
|---|---|---|
| **Оплачено** | Сделка завершена, оплата получена | ≈65% |
| **В работе** | Сделка в процессе выполнения | ≈20% |
| **Отменено** | Сделка отменена | ≈10% |
| **Отозвано** | Сделка отозвана клиентом | ≈5% |

Вероятность статуса зависит от **типа клиента** и **категории товара**.  
Например: оптовые клиенты при покупке труб оплачивают с вероятностью 82%, розничные — 55%.

## 2. Таблица «Планы продаж» (plans.csv)

Содержит плановые показатели по каждому менеджеру на каждый месяц. 30 записей (5 менеджеров × 6 месяцев).

| № | Поле | Тип данных | Описание | Пример |
|---|---|---|---|---|
| 1 | **ID менеджера** | Число | Уникальный идентификатор менеджера | 1 |
| 2 | **Месяц** | Дата | Плановый месяц | 01.03.2026 |
| 3 | **Плановая сумма** | Число | План продаж на месяц, Br | 2 850 000 |

### Особенности планов

- Планы учитывают **сезонность**: максимальные в марте и июне (конец квартала/полугодия), минимальные в январе
- Для каждого менеджера установлен индивидуальный базовый план
- Допускается случайное отклонение ±5% для реалистичности

## 3. Таблица «Клиенты» (clients.csv)

Справочник клиентов компании. Содержит информацию о каждом клиенте и его активность за весь период. 178 записей.

| № | Поле | Тип данных | Описание | Пример |
|---|---|---|---|---|
| 1 | **ID клиента** | Число | Уникальный идентификатор клиента | 42 |
| 2 | **Дата первой сделки** | Дата | Дата первой покупки клиента | 12.01.2026 |
| 3 | **Название клиента** | Текст | Наименование организации или ФИО | ООО «СтальПром-042» |
| 4 | **Тип клиента** | Текст | Опт / Розница | Опт |
| 5 | **ID менеджера (первая сделка)** | Число | ID менеджера, заключившего первую сделку | 1 |
| 6 | **Менеджер (первая сделка)** | Текст | ФИО менеджера первой сделки | Иванов А.А. |
| 7 | **Статус клиента** | Текст | Новый / Старый | Старый |
| 8 | **Всего сделок** | Число | Количество всех сделок клиента | 5 |
| 9 | **Общая сумма** | Число | Сумма всех сделок клиента, Br | 177 968 |
| 10 | **Оплачено** | Число | Количество оплаченных сделок | 3 |
| 11 | **Отменено** | Число | Количество отменённых сделок | 1 |

### Статусы клиентов

| Статус | Описание | Доля |
|---|---|---|
| **Новый** | Клиент совершил только одну покупку за всю историю | ≈14% |
| **Старый** | Клиент совершил две и более покупок | ≈86% |

## Связи между таблицами

```
sales.ID_клиента ──── clients.ID_клиента
sales.ID_менеджера ── plans.ID_менеджера
sales.ID_менеджера ── clients.ID_менеджера_первой_сделки
```

Каждая сделка в `sales` привязана к:
- Клиенту (через ID_клиента)
- Менеджеру (через ID_менеджера)

Планы в `plans` привязаны к менеджеру (через ID_менеджера) и месяцу.

## Параметры генерации

| Параметр | Значение |
|---|---|
| Количество сделок | 2 000 |
| Период | Январь — Июнь 2026 |
| Количество менеджеров | 5 |
| Количество клиентов | ~178 |
| Доля оптовых клиентов | 35% |
| Доля розничных клиентов | 65% |
| Категории товаров | Трубы, Лист, Арматура, Метизы |
| Распределение активности | Pareto (20% клиентов → 80% продаж) |
